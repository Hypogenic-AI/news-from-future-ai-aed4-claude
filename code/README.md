## How to Re-Clone

All repositories were cloned with `--depth 1` for efficiency:

```bash
cd /workspaces/news-from-future-ai-aed4-claude/code/
git clone --depth 1 https://github.com/andyzoujm/autocast.git
git clone --depth 1 https://github.com/dannyallover/llm_forecasting.git
git clone --depth 1 https://github.com/forecastingresearch/forecastbench.git
git clone --depth 1 https://github.com/TZWwww/PROPHET.git
git clone --depth 1 https://github.com/Metaculus/forecasting-tools.git
git clone --depth 1 https://github.com/Polymarket/agents.git polymarket-agents
git clone --depth 1 https://github.com/jlab-nlp/Future-Language-Modeling.git
git clone --depth 1 https://github.com/gnosis/prediction-market-agent.git
git clone --depth 1 https://github.com/shmsw25/FActScore.git factscore
git clone --depth 1 https://github.com/frazane/scoringrules.git
```
